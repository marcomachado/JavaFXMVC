O que faz foi feito anteriormente:
    * Criação da tela principal da aplicação com um menu e um espaço para rendirazação das demais telas
    * criação da tela de clientes com um TableView na lateral esquerda e um conjunto de Label's na direita para mostrar os dados.
    * Botões para adicionar, alterar e remover

Código no GitHub: https://github.com/marcomachado/JavaFXMVC

Atividade para o dia 29/08/2018 (penúltima aula de JavaFX)

1. Criar um banco no MySQL chamado javafxmvc, criar a tabela clientes de acordo com o script SQL (script_bd.sql) que está na pasta src

2. Copiar o pacote javafxmvc.model.dao com as classes ClienteDAO e ConnectionFactory. Configurar o nome de usuárioa e senha no ConnectionFactory

-- Daqui para frente só trabalharemos no controlador FXMLCadastroClienteController --

3. Criar no controlador FXMLCadastroClienteController.java os ids correspondentes à TableView e cada uma de suas colunas (TableView<Cliente> tvClientes, TableColumn<Cliente, String> tvColunaClientesNome, TableColumn<Cliente, String> tvColunaClientesCpf) -> a String irá representar o nome ou cpf do cliente, juntamente com o objeto Cliente correspondente.

4. Criar o atributo List<Cliente> listClientes -> armazenará a lista de clientes já cadastrados no banco

5. Criar o atributo ObservableList<Cliente> observableListClientes -> Lista mostrada no TableView. Armazenará a lista de clientes trazida no banco (listClientes) e adicona um observador. Esse observador faz com a lista na tela seja atualizada sempre que um novo cliente for cadastrado.

6. Ler comentários no método initialize(URL url, ResourceBundle rb) e modificá-lo

7. Ler comentários no método preencherTableViewClientes; criá-lo no projeto

8. Criar o método selecionarItemTvClientes(Cliente cliente)

9. Deixem esse código todo funcional (abrir a tela de cadastro de cliente, selecionar um cliente e mostrar seus dados ao lado trazendo tudo do banco de dados)

10. Na aula do dia 31/08 voltaremos para ele e fazemos as funcionalidades e tela para adiconar, alterar e remover clientes
